#CUET PG Quiz System 📚
A console-based quiz application built in C for CUET PG preparation.

##Features
-Multiple topics (DS, OS, DBMS, -Networking, Algorithms, Math)
-Difficulty levels (Easy, Medium, Hard, Mixed)
-Time limit for each question
--Hint option during quiz
Topic-wise performance tracking
-Leaderboard with saved scores
-Review wrong answers

##How to run
clang main.c -o quiz
./quiz
